# Adam Wu - Portfolio

Personal portfolio website showcasing my work in biology, music, and film.

**Live site:** [adamwu.github.io/portfolio](https://adamwu.github.io/portfolio/)

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Anime.js

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
